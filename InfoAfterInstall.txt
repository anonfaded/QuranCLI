 How to Run QuranCLI:
--------------------------------------------------

You can start QuranCLI in the following ways:

 * From your Start Menu (Look for a "QuranCLI" entry).
 * Using the Desktop shortcut (if you chose to create one during setup).

Once the application starts in its terminal window:

 * Type 'info' and press Enter to see a detailed list of commands and credits.
 * Type 'quit' or 'exit' and press Enter to close the application.

--------------------------------------------------
 Subtitle Files Location:
--------------------------------------------------

 * Remember, any subtitle (.srt) files you generate using the 'sub' command will be saved in your Documents folder, inside a directory named "QuranCLI Subtitles".
